Initializes a simulation. The function wraps the user's script into a Lua coroutine and assignes a user defined environment to the loaded script. The simulation is halted for a few milliseconds in order to poll the GUI.

Parameters

  • script: Script text to be executed.
  • nogui: Indicate whether this function is called via the GUI or not.
