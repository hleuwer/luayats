mux:init (param)
Constructor for class 'mux'. The multiplexer provides 'ninp' inputs with a single output queue with a given maximum length 'buff'. Incoming cells are enqueued by fair strategy (random choice). During each tick a single cell is scheduled. The multiplexer maintains the following loss counters:
a) for the queue in total and
b) on a per VCI level.

Parameters

  • param: table - parameter list
    • name (optional)
      Name of the display. Default: "objNN"
    • ninp
      Number of inputs
    • buff
      Size of the output buffer
    • maxvci (optional)
      Maximum value for input VCI. Default: ninp+1
    • out
      Connection to successor Format: {"name-of-successor", "input-pin-of-successor"}
    .

Return value:

table - reference to object instance.
muxAF:init (param)
Constructor for class 'muxAF'. The multiplexer provides 'ninp' inputs with a single output queue with a given maximum length 'buff'. Incoming cells are enqueued by fair strategy (random choice). Server time is configurable. Note that muxAF does not imply the additional delay as the network object yats.mux. The multiplexer maintains loss counters a) for the queue in total and
b) on a per VCI level.

Parameters

      Name of the display. Default: "objNN".
    • ninp
      Number of inputs.
    • buff
      Size of the output buffer.
    • maxvci (optional)
      Maximum value for input VCI. Default: ninp+1.
    • active
      Server time distance.
    • out
      Connection to successor Format: {"name-of-successor", "input-pin-of-successor"}.
    .
