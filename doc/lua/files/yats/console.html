Idle thread. It is activated when the simulation pauses. During paused simulation the idle thread acts on commands [[go]] and [[reset]]. The worker thread is resumed when simulation continoues.

Intercept simulation cyclically. Called by a luacontrol object and implements the states [[run]] and [[reset]].

Parameters

  • self: table Reference to luacontrol object.
  • action: table Action to perform.
  • ev: userdata Event that caused interception.
  • when: number Current simulation timeslot.
