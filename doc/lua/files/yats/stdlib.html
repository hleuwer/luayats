File yats/stdlib.lua

Yats' version of a Lua Library extension.

module: G

The 'yats.stdlib' module defines several useful functions. All of them are in the global namespace. See list of functions for details.

NOTE:Note, that yats.stdlib is not the same as stdlib from luarocks.
