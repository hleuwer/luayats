Luayats

YATS is a small discrete-time simulation tool tailored for investigations of ATM and packet networks. It uses Lua as a script language for the problem description and for rapid prototyping.

License

Luayats is released under GPL.

Features

  • Time discrete simulation for traffic management aspects of ATM and packet networks
  • Protocol simulation 
  • Statistic evaluation of simulation results, such as confidence levels and batch means
  • Fast prototyping of network nodes without re-compilation using Lua scripting
  • Hierarchical netlist creation
  • Build-in support for flow control and feedback channels.
  • Fast C++ based simulation Kernel
  • Lua is used as the interface between the user and the kernel and for problem description
  • Graphical User Interface for editing, running and observing the simulation
  • Very fast graphical display of rates, histograms

    Installation

    Luayats can be installed on Linux and Windows platforms. On Windows the Cygwin toolkit is required for all components.

    You need the following components to compile and install Luayats:

    • Lua version 5.1 (=> 5.1.4)
    • tolua++-1.0.92  binding generator
    • IUP 3.0rc3 GUI tookit
    • CD 5.2 canvas draw (2D) library

    During runtime the following loadable Lua modules are required during runtime:

    • bitlib-23.2 - bitwise operations in Lua
    • readline-1.0.1 - a simple binding to the GNU readline library
    • toluaex-1.0 - useful extension for tolua++
    • IUP Lua modules (version 3.0rc3) - comes with IUP-3.0
    • CD Lua modules (version 5.2) - comes with CD-5.2
    • luafilesystem-1.4.2 - Lua filesystem

    For detailed install instructions see the README file that comes with the distribution.
