Statistics Classes and Objects

Luayats provides a class to calculate the confidence level of simulation results.

See here for an example. 

Luayats file: statist.lua

Network ObjectClassDescription
Confidence Intervalyats.confidCalculate confidence values of simulation results. The object is instantiated like a normal network object, but does not receive or send any events to other objects. However, values are collected in a dynamic array and the running confidence level and a couple of statistical values like upper and lower bounds, interval width, variance and so on can be retrieved.
